<h2>👆 Como instalar</h2>

```bash
  # Clone o projeto e entre no diretório
  $ git clone https://github.com/pauloelf/buddys-life.git
  $ cd buddys-life
```
```bash
  # Instale as dependências
  $ npm install
```
```bash
  # Inicie o projeto
  $ npm run dev
```
